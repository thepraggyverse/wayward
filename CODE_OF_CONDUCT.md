# Code Of Conduct

Wayward is a small open-source project, and we want it to stay practical, respectful, and welcoming.

## Expected Behavior

- Be kind and direct.
- Assume good intent, then ask clarifying questions when something is unclear.
- Keep criticism focused on code, docs, behavior, and safety tradeoffs.
- Respect contributor time by keeping issues and pull requests specific.
- Credit ideas, prior work, and reported issues when appropriate.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Doxxing or sharing private information.
- Sexualized or violent content in project spaces.
- Repeatedly derailing issues, pull requests, or reviews after maintainers ask for focus.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, reviews, and any official community spaces for Wayward.

## Reporting

If you see behavior that violates this code of conduct, contact the maintainers through the repository's private reporting channel when available. If private reporting is not available, open a minimal public issue asking maintainers to establish a private contact path without posting sensitive details.

## Enforcement

Maintainers may remove comments, close issues or pull requests, block users, or take other reasonable steps to keep the project healthy. Enforcement should be proportionate, documented when appropriate, and focused on restoring a safe working environment.
